# CV / Curriculum Vitae

## Instrucciones

Coloca tu CV en formato PDF en esta carpeta con el nombre:

```
Yeison_Moreno_CV.pdf
```

## Requisitos

- **Formato:** PDF
- **Tamaño máximo:** 2MB
- **Nombre del archivo:** Exactamente `Yeison_Moreno_CV.pdf` (respeta mayúsculas/minúsculas)

## Cómo Crear un CV Profesional

### Herramientas Recomendadas:

1. **Canva** (Gratis)
   - https://www.canva.com/
   - Plantillas profesionales de CV
   - Fácil de usar

2. **Novoresume** (Gratis con limitaciones)
   - https://novoresume.com/
   - Templates específicos para tech

3. **Resume.io** (Gratis con limitaciones)
   - https://resume.io/
   - Muy usado en la industria

4. **LaTeX Overleaf** (Gratis, avanzado)
   - https://www.overleaf.com/
   - Para CVs técnicos muy profesionales
   - Plantilla recomendada: "Jake's Resume"

### Secciones que Debe Tener tu CV:

1. **Header**
   - Nombre completo
   - Título profesional
   - Email, teléfono, LinkedIn, GitHub
   - Ubicación

2. **Summary/Profile**
   - 2-3 líneas sobre ti
   - Enfócate en logros y skills clave

3. **Experience**
   - Empresas y fechas
   - Cargo
   - 3-5 bullets de logros (no responsabilidades)
   - Usa números y métricas

4. **Skills**
   - Technical Skills
   - Soft Skills
   - Certifications

5. **Education**
   - Universidad
   - Título
   - Año

6. **Certifications (opcional)**
   - Platzi, Coursera, etc.

### Tips:

- ✅ Usa formato ATS-friendly (que los robots puedan leer)
- ✅ Usa verbos de acción: "Desarrollé", "Lideré", "Optimicé"
- ✅ Incluye números: "60% reducción", "10+ procesos"
- ✅ Máximo 2 páginas
- ✅ Sin foto (a menos que sea común en tu país)
- ❌ No uses tablas complejas
- ❌ No uses colores muy llamativos
- ❌ No incluyas información personal sensible

## Verificación

Para verificar que tu CV está bien colocado:

1. Abre `index.html` en tu navegador
2. Busca el botón "Descargar CV"
3. Click en el botón
4. Debería descargar tu PDF

Si no funciona:
- Verifica que el archivo esté en esta carpeta
- Verifica el nombre del archivo (exactamente `Yeison_Moreno_CV.pdf`)
- Verifica que sea un PDF válido
