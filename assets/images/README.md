# Imágenes del Portfolio

Esta carpeta contiene todas las imágenes necesarias para el portfolio.

## Estructura de Carpetas

```
assets/images/
├── profile.jpg          # Tu foto profesional
├── hero-bg.jpg         # Imagen de fondo para hero section (opcional)
└── logos/              # Logos de empresas
    ├── accenture.png   # Logo de Accenture
    ├── ups.png         # Logo de UPS
    └── platzi.png      # Logo de Platzi
```

## Imágenes Requeridas

### 1. Foto de Perfil (profile.jpg)
- **Tamaño recomendado:** 500x500px
- **Formato:** JPG o PNG
- **Descripción:** Tu foto profesional para la sección "About Me"
- **Dónde conseguirla:** Usa una foto profesional tipo LinkedIn

### 2. Logos de Empresas

#### Accenture Logo
- **Archivo:** logos/accenture.png
- **Tamaño:** 200x200px (se ajustará automáticamente)
- **Descarga desde:** https://www.accenture.com/us-en/about/company/media-resources
- O busca "Accenture logo PNG transparent"

#### UPS Logo
- **Archivo:** logos/ups.png
- **Tamaño:** 200x200px
- **Descarga desde:** https://www.ups.com/us/en/about/news/brand.page
- O busca "UPS logo PNG transparent"

#### Platzi Logo
- **Archivo:** logos/platzi.png
- **Tamaño:** 200x200px
- **Descarga desde:** https://static.platzi.com/media/platzi-isotipo.png
- O busca "Platzi logo PNG transparent"

## Placeholder Temporal

Si aún no tienes las imágenes, puedes usar placeholders temporales:

### Opciones para Placeholders:
1. **https://placeholder.com/** - Placeholders simples
2. **https://picsum.photos/** - Fotos random
3. **https://ui-avatars.com/** - Avatares generados

### Ejemplos:
```
Profile: https://ui-avatars.com/api/?name=Yeison+Moreno&size=500&background=00D9FF&color=fff
Accenture: https://logo.clearbit.com/accenture.com
UPS: https://logo.clearbit.com/ups.com
Platzi: https://static.platzi.com/media/platzi-isotipo.png
```

## Optimización de Imágenes

Antes de subir tus imágenes, optimízalas para web:

### Herramientas Recomendadas:
- **TinyPNG:** https://tinypng.com/ (reduce tamaño sin perder calidad)
- **Squoosh:** https://squoosh.app/ (optimizador de Google)
- **ImageOptim:** https://imageoptim.com/ (Mac)

### Consejos:
- Mantén los archivos bajo 200KB
- Usa formato PNG para logos (transparencia)
- Usa formato JPG para fotos
- Considera WebP para mejor compresión

## Notas

- Los logos deben ser en formato PNG con fondo transparente
- Asegúrate de tener derecho a usar los logos de las empresas
- La carpeta `logos/` debe contener todos los logos antes del deploy final
